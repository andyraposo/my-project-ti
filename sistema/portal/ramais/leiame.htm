Orientações de uso dos códigos do DataGrid

Requisitos técnicos:
            Um servidor web que interprete o PHP;
            O banco de dados MySQL;

Conhecimentos necessários para implementação dos códigos:
            Uso de navegadores (Internet Explorer, Mozila, Safari...);
            Noções de PHP;
            Noções de JavaScript;
            Noções de uso do PHPMyAdmin;

Público-alvo:
            Desenvolvedores e Analistas de Sistemas que utilizem a linguagem PHP;
            Designers e curiosos que queiram entender como se dá o processo de inclusão, alteração e exclusão de dados, utilizando MySQL, PHP e AJAX;

DataGrid: é uma tabela onde estão exibidos os produtos cadastrados em um banco de dados, onde se é possível fazer a inclusão, alteração e exclusão dos produtos cadastrados pela própria tabela, sem a necessidade de abrir páginas e formulários secundários para este fim.
            O DataGrid é composto pelos seguintes arquivos:

  1. bibliotecaAjax.js: é composto por um conjunto de funções que tratam e manipulam as informações enviadas do browser ao servidor e vice-versa;
  2. estiloDataGrid.css: contém os estilos que serão utilizados pelo sistema;
  3. sugestoes.php: Trata os dados enviados pelo browser e retorna os valores extraídos do banco de dados em formato XML;
  4. index.php: Contém os códigos que exibirão os produtos existentes no banco de dados em forma de tabela e os controle necessários a sua manipulação;
  5. index2.php: Contém os códigos que farão a inclusão, alteração e exclusão dos produtos;
  6. banco_de_dados.txt: Contém as instruções em SQL que criarão o banco de dados, a tabela e fará a inserção de dados que serão usados como exemplo, mas que podem ser apagados posteriormente;

Instalação:

  1. Abra o arquivo banco_de_dados.txt e copie todas as informações contidas nele;
  2. No PHPMyAdmin, clique em banco de dados e em seguida em SQL;
  3. Cole as informações que você copiou do arquivo banco_de_dados.txt no campo de texto apresentado;
  4. Em seguida clique em executar;
  5. Copie os arquivos bibiotecaAjax.js, estiloDataGrid.css, index.php e index2.php para uma pasta do seu servidor web;
  6. Abre um navegador qualquer e digite a url: http://<nome_do_servidor>/ <nome_da_pasta>/ index.php;
  7. Se abrir a tela com a tabela de produtos, o sistema estará pronto pra você usar. Caso apresente erro de exibição, verifique se a url está correta.
